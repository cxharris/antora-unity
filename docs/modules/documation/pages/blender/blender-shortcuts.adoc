= Blender shortcuts
:navtitle: Keyboard shortcuts

An in-progress list of all the keyboard shortcuts we've encountered.

[cols="1,1,1,1"]
|===
| Shortcut | Meaning | Notes | Window/Mode

| A        | Select All | selects all components of an object when perhaps just one component (e.g. a vertex) might be selected | Edit
| ALT - A  | Deselect All | In edit mode, deselects the entirety of a selected object. | Edit
| CTRL - B | Bevel  | Bevel edges of selected object | Edit
| CTRL - L | Link  | Offers options to link a chain of selected objects | Object
| CTRL - R | Loop cut  | Creates a loop of one or more edges along a ring of quads (even one quad!) | Edit
| E        | Extrude    | Extrude a selected face - by default along the face normal. | Edit
| G        | Grab    | Grab/move a currently-selected entity. | Object or edit
| G - X,Y,Z | Constrained grab | Constrain the grab to make the object move on the X, Y or Z axes respectively. Hitting SHIFT as well makes the move happen in smaller increments. | Object or edit - in edit mode, pivot is independent
| MIDDLE MOUSE | Tumble | Tumble around a scene | Object or edit
| NUMPAD 0 | Camera view | Presents a view through the camera | Object or edit
| R        | Rotate    | Rotate a currently-selected entity about the 3D cursor (... possibly other options) | Object or edit
| SCROLL WHEEL | Zoom  | Zoom into or out of the scene | Object or edit
| SHIFT - A | Add | Opens the *Add* menu | Object
| SHIFT - LEFT ARROW | Reset animation | Reset current animation frame to zero | Timeline window
| SHIFT - MIDDLE MOUSE | Pan | Pan around a scene | Object or edit
| SPACEBAR | Play | Play an animation | Timeline window
| Z | Visual Mode | Allows flipping between wireframe, solid, material preview or rendered modes | Object or edit
|=== 

There's a really complete set of keybindings here: https://www.katsbits.com/tutorials/blender/useful-keyboard-shortcuts.php
